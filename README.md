# Games Collection

Welcome to my collection of mini-games! This repository is a personal playground where I experiment with game logic, terminal UI, and programming languages. It features a mix of text-based RPGs, classic puzzles, and logic games built using **C++** and **Python**.

---

## Featured Games

### C++ Games 
*   **[PokemonGame1.cpp](PokemonGame1.cpp)**: A text-based Pokémon experience. Turn-based combat mechanics featuring classic battle logic.
*   **[detective_RPG.cpp](detective_RPG.cpp)**: An immersive text-driven RPG where you gather clues, make choices, and solve a mystery.
*   **[rewind_maze.cpp](rewind_maze.cpp)**: A unique puzzle maze game with a "rewind" mechanical twist to challenge your navigation skills.
*   **[TicTacToe.cpp](TicTacToe.cpp)**: The classic 3x3 grid game optimized for two players right in the console.
*   **[room12.cpp](room12.cpp)**: A text adventure room-escape style script testing player logic and choices.

### Python Games 
*   **[Rock Paper Scissors.py](Rock%20Paper%20Scissors.py)**: The timeless hand game played against an automated computer opponent.
*   **[guessGame.py](guessGame.py)**: A number-guessing game using randomized logic and feedback hints (higher/lower).

## How to Run the Games

### Running C++ Games
Ensure you have a C++ compiler installed.

1. Open your terminal and navigate to the project directory.
2. Compile your game of choice (e.g., TicTacToe):
   ```bash
   g++ TicTacToe.cpp -o TicTacToe


### Running Python Games
Ensure you have Python 3 installed.

1. Run the script directly from your terminal:
  ```bash
  python "Rock Paper Scissors.py"
